Projects

I've created this page to try and provide links to specific projects I have worked on. It is difficult to try and capture lists of projects from the past, I find documenting them in real-time is the best way.  As I bring projects live, make updates, or even archive a project I will list here and try to summarize.

2013

  • I Want An API - I want to outline what it takes to deploy and API either with a cloud service or on your open using open source tools.
  • Digital Strategy Scorecard - A new attempt to track on the White House Digital Strategy using the Hacker Storytelling format and produce visualizations showing progress, but using the APIs and data available as part of the digital strategy.
  • Hacker Storytelling - This is my new project platform for 2013.  Allowing me to make every project I do, be open and machine readable by default.  As you can see some of the projects below don't have links anymore, because they went away.  With Hacker Storytelling they will stick around longer.
  • IPA Evangelist - This is an ongoing project documenting the darker side of my personality, as an IPA evangelist.
  • The API Stack - A weekly dump of the top 25 APIs I'm tracking on each week, with news and analysis.

2012

  • Hack Weekends Guide - This is my mashup of several how to throw a hackathon guides into a single, open source implementation complete with template hackathon website.
  • Earth as Art - I took an amazing collection of earth photos from NASA, liberated from PDF and published as project on Github, with all the images and JSON feed, so anyone can remix.  It was my first Hacker Storytelling project.
  • Business of Twitters API - I just wanted to understand how Twitter and its API evolved and changed since they launched in 2006, allowing me to have a better understanding of what Twitter has done right, but also what it did wrong.
  • White House Digital Strategy - I wanted to track on the progress of the White House digital strategy.
  • Hack Weekends - My information, calendar and news site for the Hackathon space.

2011

  • Elance Powered API Developer Area - This is a prototype to demonstrate how to integrate Elance developer community into an API community.  Unfortunately this site isn't up anymore either.  
  • Federal Budget Prioritization - I deployed an older project and merged it with data from Obamas 2012 proposed budget.  I wanted users to be able to prioritize their budget priorities.  I did this for state of Oregon and California in 2010 and federal level in 2012.  Unfortunately I took the sites down.
  • Alternate Kin Lane - Where I put my fictional work, to excercise my creativity and not be swallowed by the tech blogosphere.
