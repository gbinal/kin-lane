{% include JB/setup %}<p>Google released a new Google Books API this week.The new Books API will allow developers to build mobile and web applications using the more than 15 million books titles available on books.google.com.The Books API allows searching of books metadata, pricing and working with users Google Book accounts to retrieve and manage their bookshelves and the titles they contain.</p>
