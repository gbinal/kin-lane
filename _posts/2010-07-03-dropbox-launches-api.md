    desktop.The API offers the following features: Simple HTTP+JSON method of accessing a users information in a user approved sandbox on the users desktop.List, upload, delete, move, copy, and get
    files as well as many other features.A full event callback API allowing you to get simple asynchronous events to your web site or service whenever the user changes their sandbox.
