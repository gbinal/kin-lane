    TIFF ImageMagick can scale, flip, mirror, rotate, distort, shear and transform images, adjust image colors, apply various special effects, or draw text, lines and shapes ImageMagick is a command
    line utility which you can choose from several libraries: JMagick for Java MagickNet for.NET MagickWand for PHP IMagick for PHP PythonMagick for Python RMagick for Ruby I spent about an hour
