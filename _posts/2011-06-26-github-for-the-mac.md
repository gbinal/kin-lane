    Github just released a new Github client for the Mac.It provides a GUI alternative to the Github command line.Some features of the new Github for Mac are: Synchronize Branches - The sync button
    pushes your changes to GitHub and pulls down others changes in one operation, and notifies you when you have changes you havent pushed or there are new changes on GitHub you havent pulled
    down.Clone Repositories - When you add repositories to GitHub for Mac, it automatically matcesh them up with any organizations you belong to.Simpler Branching - Allows you to try out remote
    branches, create new local branches and publish branches to share with others.
