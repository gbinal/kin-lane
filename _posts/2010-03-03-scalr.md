    What is Scalr?Scalr is software that creates redundant, self-curing, and auto-scaling clusters to run your website.Using EC2 and Cloud Computing.It allows you to create server farms through a
    web-based interface using prebuilt AMIs for load balancers (pound, nginx, or Amazons load balancing service), app servers (apache, rails, others), databases (mysql master-slave, others), and a
    generic AMI to build on top of.The health of the farm is continuously monitored and maintained.
